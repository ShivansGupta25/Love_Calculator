# ❤️ Love Calculator

A simple and fun Love Calculator web app built using HTML, CSS, and JavaScript.

🚀 This project calculates the "love compatibility percentage" between two names and displays the result instantly.

---

## ✨ Features

- ❤️ Calculates love compatibility between two names
- ⚡ Instant result using JavaScript
- 🎨 Simple and clean UI
- 💻 Beginner-friendly project
